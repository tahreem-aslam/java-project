#OOP Java Project
## Subject 
Object Oriented Programming
## Language
Java
## Description
This is Community Skill Sharing Platform which is object-oriented desktop application built with java and javaFX. It connects Skill Providers(Teachers) with Skill Seekers (Students) allowing the providers to offer courses and seekers to enroll in them. This simulation shows how a community-based education system works using simple Object-Oriented Programming (OOP) principles like Encapsulation, Inheritance, Abstraction, and Polymorphism

Run Main.java file
